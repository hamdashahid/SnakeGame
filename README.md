# Snake Game

A classic Snake game implemented in C++ using OpenGL.

## Features
- Grid-based movement
- Randomly generated food items
- Hurdles for added challenge
- Score tracking
- Game over screen
- Main menu for starting or exiting the game

## Controls
- **Arrow Keys**: Move the snake (Up, Down, Left, Right)
- **A**: Start a new game
- **R**: Return to the main menu
- **Esc**: Exit the game

## Installation & Compilation
### Requirements:
- C++ compiler (G++)
- OpenGL and GLUT libraries

### Compilation (Linux/macOS)
Run the following command in the terminal:
```sh
g++ game-release.cpp -o snake_game -lGL -lGLU -lglut
./snake_game
